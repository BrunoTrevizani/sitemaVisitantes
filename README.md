# 📝 Sistema de Visitantes

Um sistema simples de **cadastro e controle de visitantes**, desenvolvido em **PHP** com **MySQL**.  
Permite login de usuários, registro de acessos e gerenciamento básico de informações.

---

## 🚀 Tecnologias usadas
- PHP 8+
- MySQL/MariaDB
- HTML, CSS
- Apache (XAMPP/LAMP)
